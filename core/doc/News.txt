The Related Values Processing Framework (RVPF) is being developed to
help the integration of Process Control Data Historian Systems. This is
the eighth files release of RVPF.

The software is written in Java. All libraries needed to use this
software are available under an open source license.

RVPF is licensed under the GNU Lesser General Public License (LGPL)
version 2.1.

The project's home page is located at http://rvpf.org/.
