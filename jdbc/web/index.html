Overview

The RVPF-JDBC child project provides a JDBC driver to present a well known interface to access the content managed by a RVPF Store service.

Context

A RVPF Store service manages manufacturing points data. This data may be seen as a simple table where each row contains a reference to a point definition, a timestamp, a state and a value. The Store service also has read access to the relationship between point names and the UUID used as the point definition reference.
